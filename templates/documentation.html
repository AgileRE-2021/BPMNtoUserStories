Documentation

How to use :

1. Enter the home page, then click "See More"

2. Then press “Choose File”

3. Then select the BPMN file that you want to translate to User Stories (filetype must be .XML / .XPDL) and Press "Open"

4. If you have finished uploading the appropriate file, then press "Submit"

5. After clicking Submit, your BPMN file is already seen turning into User Stories
