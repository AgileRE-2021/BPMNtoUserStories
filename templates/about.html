BPMN To User Stories

Dibangun oleh mahasiswa Sistem Informasi

Universitas Airlangga

Aplikasi BPMN to User Stories merupakan sebuah aplikasi berbasis web dimana

aplikasi ini dapat melakukan translasi dari BPMN ( filetype : .XML / .XPDL )

menjadi User Stories. User dapat mengupload file BPMN dengan tipe file yaitu .XML atau .XPDL,

kemudian user dapat melakukan generate dari file BPMN tadi menjadi User Stories dengan menekan tombol generate.

Selain itu, aplikasi ini juga dapat memperlihatkan history setiap user melaukan generate.
